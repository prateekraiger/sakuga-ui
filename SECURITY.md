# Security Policy

## Supported Versions

Only the latest version of Sakuga UI is supported for security updates.

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please do not report it publicly through GitHub issues. Instead, please email the author directly at:

**Email:** prateekraiger123@gmail.com

We will acknowledge receipt of your vulnerability report within 48 hours and work with you to resolve the issue as quickly as possible.
